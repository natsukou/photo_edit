# 📷 AI拍照辅助工具

一款基于AI的智能拍照辅助工具，帮助摄影小白和内容创作者快速掌握拍摄技巧。

## ✨ 功能特性

- 🤖 **AI智能识别**：自动识别照片题材和风格
- 📐 **辅助线工具**：九宫格、三分线、黄金分割等构图辅助
- 💡 **拍摄建议**：根据照片分析提供专业的构图、光线、角度建议
- 🎨 **风格推荐**：支持10+拍摄题材和10+风格选择

## 🚀 使用方法

1. 上传或拍摄照片
2. AI自动识别题材和风格
3. 获取专业拍摄建议
4. 应用辅助线优化构图

## 🛠️ 技术栈

- 前端：HTML + CSS + JavaScript
- AI识别：阿里云通义千问
- 部署：阿里云函数计算 (Serverless)
